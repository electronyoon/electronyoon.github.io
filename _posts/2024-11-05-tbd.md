### 가상머신의 등장

예를 들어, Java 코드는 이렇게[^1]
```
// 고수준
System.out.println("Hello, World!");

// 저수준(바이트코드)
0: getstatic               #2
3: ldc                     #3
5: invokevirtual           #4
8: return
```

Python 코드는 이렇게[^2] 번역됩니다.
```
// 고수준
print("Hello, World!")

// 저수준
0 LOAD_NAME                0 (print)
2 LOAD_CONST               1 ('Hello, World!')
4 CALL_FUNCTION            1
6 RETURN_VALUE
```



---

[^1]: javap 명령어 사용
[^2]: dis.dis() 모듈 사용
